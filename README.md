## ▶️ Cómo ejecutar

1. Instalá pytest (si no lo tenés):
   ```bash
   pip install pytest
   ```

2. Ejecutá los tests desde terminal:
   ```bash
   python -m pytest
   ```

O directamente desde `run_tests.py`:

```bash
python run_tests.py
```
